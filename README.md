# Procure.ai — Enterprise Procurement Intelligence

This workspace contains a React + Tailwind prototype for an enterprise procurement analytics website.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the URL shown in the terminal to see the website.

## Project Structure

- `src/App.jsx` — main website shell and navigation
- `src/components/ProcurementAssistant.jsx` — enterprise procurement assistant UI
- `src/main.jsx` — React entrypoint
- `src/styles.css` — Tailwind styles
- `package.json` — app dependencies and scripts

## Notes

The current implementation is a frontend prototype. Backend services for contract analysis, RFQ generation, and spreadsheet queries can be integrated via API endpoints in the next phase.
